Chaotic Cats

2025 Game Development

Overview

Chaotic Cats is a 2D game built in Unity where players control mischievous cats causing havoc in a cartoon-style world. I led core gameplay development as part of a team project, taking ownership of the systems that made the game feel good to play.

The focus was on tight, responsive controls and a variety of mechanics — including a shark-avoidance mini-game that injected tension into the otherwise playful tone. Working in an agile team meant shipping iteratively, playtesting frequently, and refining based on feedback.

What I built

  • Core player controller and movement system
  • Collision detection and physics-based interaction system
  • Shark avoidance mini-game with escalating difficulty
  • Character animations and sprite state management
  • Version-controlled workflow across the team using Git

Tech stack

Unity EngineC# 2D AnimationSprite Management OOPGitAgile

Reflection

Game development taught me how much of software engineering is about feel and iteration. A system can be technically correct and still feel wrong — the gap between the two is something you only close through playtesting and the willingness to throw things out and start again.

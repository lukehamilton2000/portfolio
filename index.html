Autonomous Flower Detection System

Project Overview

This MSc dissertation project focused on developing an innovative AI-powered computer vision system for autonomous flower detection and tracking in precision agriculture applications. The system demonstrates the practical application of artificial intelligence in addressing challenges in sustainable agriculture.

Technical Implementation

The core of the system utilizes YOLOv8 with custom transfer learning optimizations to achieve real-time object detection performance on resource-constrained hardware. The machine vision pipeline was designed to be robust against various environmental challenges including:

  • Variable lighting conditions
  • Partial occlusion
  • Different viewing angles
  • Environmental factors

Innovation & Impact

This project represents a significant step forward in precision agriculture, offering potential solutions for:

  • Automated pollination systems
  • Crop monitoring and yield prediction
  • Sustainable farming practices
  • Integration with robotic systems
